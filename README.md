# Smart-Garden

## Sobre o Projeto

**Projeto:** Smart-Garden 

**Problema que resolve:** O Smart Garden resolve o problema da irrigação ineficiente de plantas, evitando tanto o excesso quanto a falta de água por meio da integração de sensores de umidade do solo com dados de previsão do tempo. Utilizando inteligência artificial, o sistema analisa essas informações em tempo real para decidir automaticamente se a irrigação deve ser realizada ou adiada, garantindo o uso consciente de água, reduzindo desperdícios e facilitando o cuidado com plantas mesmo para usuários com pouca experiência ou tempo disponível.
